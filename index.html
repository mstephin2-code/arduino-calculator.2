💻 Arduino Calculator Project

A Simple, Yet Powerful Standalone Calculator

📜 Project Overview

This project successfully implements a basic, standalone four-function calculator (addition, subtraction, multiplication, and division) using an **Arduino UNO** microcontroller. The user interacts with the device via a **keypad**, and the results are displayed on a **16x2 Liquid Crystal Display (LCD)**. The project demonstrates fundamental concepts of microcontroller programming, external hardware interfacing, and input/output processing.

The code is designed to handle multi-digit numbers, negative numbers, and floating-point arithmetic, making it a functional and practical device. It even includes a custom splash screen!

🛠 Components Used

The following components were essential for bringing the Arduino calculator to life:

  • **Arduino UNO:** The brain of the project, executing the calculator logic.
  • **16x2 LCD Display:** Used to show user input, the operation being performed, and the calculated result. The 16x2 refers to 16 characters per line on 2 lines.
  • **4x4 Keypad:** Provides the input interface, allowing the user to enter numbers (0-9), the decimal point (.), and the four basic arithmetic operations (+, -, *, /) and the equals sign (=).
  • **Jumper Wires & Breadboard (not explicitly pictured, but often used):** For making the electrical connections between the components.

💻 Arduino Code and Logic

The project uses two primary libraries: **`LiquidCrystal.h`** for the display and **`Keypad.h`** for the input. The code implements a state-machine-like approach to handle the calculator's operations:

  1. **Initialization (`setup()`):** Sets up the LCD and displays a brief *'Arduino Calculator'* splash screen.
  2. **Input Handling:** The `loop()` continuously checks for a key press using `keypad.getKey()`.
  3. **Processing (`processInput()`):**
    • **Numbers & Decimal:** Appends digits to the `current` string. Logic is included to prevent multiple decimal points.
    • **Operators (+, -, *, /):** If an operator is pressed, the current number is moved to the `memory` variable, and the new operator is stored in the `operation` variable. The `current` variable is cleared for the second number.
    • **Equals (=):** Converts the `memory` (first number) and `current` (second number) to floats, calls the `calculate()` function, and then displays the result back to the LCD.
  4. **Display Update:** Input and results are printed directly to the LCD, maintaining the correct cursor position.
